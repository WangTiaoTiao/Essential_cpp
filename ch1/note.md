#### 7.3

* 回忆第4章， 数组
* int arr[] == int* arr 在函数头中是等价的。
* 重要的等式
    * `arr[i] == *(arr + i)`
    * `&arr[i] == arr + i`
* 传递数组 = 传递 地址 和 长度
    * 好处：节省拷贝的内存和时间
    * 问题：直接操作数组的风险； 如何避免 ：const
* 64 位系统的指针也占64位。
